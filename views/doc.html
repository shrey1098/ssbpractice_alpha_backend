SSBPractice API

SSBPractice API is a RESTful API that provides data for the SSBPractice website. The API is used to fetch data from the database and send it to the website. The API is also used to update the database with the data sent by the website.

The API is hosted on AWS and is available at https://ssbpractice.click/

The API v1.0 is in production and v2.0 is in development

Endpoints

The API has the following endpoints:

  • /wat

/wat

Get 60 words for wat

Method: GET

URL: https://ssbpractice.click/wat

Sample Response: { 1: {word:'battle'} 2:{word:'war'} }

/srt

Get 60 situations for srt

Method: GET

URL: https://ssbpractice.click/srt

Sample Response: { 1: {situation:'Situation 1'} 2:{situation:'situation 2'} }
